 精准匹配买家搜索

+ 大词/核心词
    类目词：shoes，keys，phone，shirt
+ 衍生词
    修饰大词，切中痛点：商品属性（颜色功能）、使用场景、卖点：free shipping，notebook promotions，school shoes
+ 长尾词
    多个词组成，商品属性、搜索栏推荐词：Lenovo ThinkPad X1 Carbon Ultrabook

标题结构
+ 品牌
店铺名/品牌名+（修饰词）产品名字（修饰词）+型号/尺寸（修饰词）
+ 无品牌
产品名+核心卖点+适用场景（型号/尺寸）
