前几天轮滑课的教练说有个比赛建议我们参加一下，我刚开始推脱有事，他们还是努力说服我参加，
我就直接跟他们说，这跟我个人的理念有冲突，我不想小孩子才4岁就去刻意的培养所谓竞争意识，
不想让娃过早的去感受这种氛围，然后那个教练说： 哦哦 快乐教育对吧

我沉默了下然后笑了笑，没有反驳，我内心说快乐教育早已是个负面词汇了，我并非是所谓快乐教育，真是不想拔苗助长而已，我是从一个全国贫困县努力考出来的，高考复读了两次，最后是全省40多万考生，2000名读了一个不算太差的大学，我只是从自己的人生经历总结经验，尽量的保持小孩最本真的学习动力和热情，不愿去用比赛或其他的排名去摧毁，人为什么非要去跟别人比呢，其实跟自己比才是应该做的事情，每个人有自己的轨迹和人生，有些人天生是运动员可以去比赛，我自知我不是，我的娃自然大概率也不是，所以我就是做好父母的本分，守护娃的自然成长，给予养分和支持，至少不要像我过早的失去了动力
