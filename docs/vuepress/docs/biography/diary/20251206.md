最近想起一件事情，我几年前从新加坡休假回国，回到老家参加弟弟婚礼，有个我爸的好朋友过来帮忙，是个瘦瘦小小的老头，人挺积极和善，但是聊天的时候他问我：新加坡大部分都是中国的，我说是，华人占多数，他接着说，对中国不好是坏种，我说新加坡是个独立国家有自己的政策，他又重复了一句新加坡是坏种，我尴尬的笑笑，内心想新加坡是中立的国家，真是个愚昧的老头；

但是时间转到现在，当我最近突然看到新加坡的总理关于中日关系的讲话，话里话外明显的忽略中国人的苦难，并且偏袒日本，说实话我完全无法接受以至于产生愤怒的情绪，我爷爷曾经领导过苏鲁豫皖抗日革命，我们跟日本人的仇恨不共戴天

然后我想起最近看到有人讲义和团，以前的历史书和电影把义和团描写的比较负面，终于有人从正面的角度肯定了基层人民发起的义和团行动，正是有了这些“愚昧”的基层民众，才逐渐培养了后面的正规的力量，如果没有民间朴素的爱恨情仇，直接果敢的“冲动”，早就亡国灭种了

在极端的情况下，扎根在中国大地的这些草芥百姓才是最后的堡垒，你是敌人就是敌人，坏种就是坏种，朋友就是朋友，敢来欺负我们拿起柴刀就去砍你，义和团的精神才是中华文明延续的火种
