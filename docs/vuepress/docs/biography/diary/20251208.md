思想的麻木

人真的很难一直提高警惕，大部分情况下处于放松的状态是极容易被洗脑的，
比如电梯广告一直播 小儿咳嗽就喝xxx，我并非说听多了我就回去买（事实是对我爸确实起效果了，当我的小孩某次咳嗽时，我爸脱口而出说那就买xxx），我对广告还是比较抵制和反感的，但是问题是我跟我爸也是五十步和百步的区别而已，
打个比方，所有人都说牛奶有营养，我就觉着喝牛奶就是健康的选择，但是直到我听了倪师的讲解才知道非也

作为一个活生生的个体，所有的宣传或者“常识”都不应该轻易的接受，在被迫接受西方资本消费主义的洗脑的社会环境下，至少几千年的中医是我们可以拿起的武器，不要轻易的将身体交给西医，动辄头疼脑热的就吃几片药，确实可能见效快，但是好不好呢，至少我个人的真实经历验证结果是：不好，非常不好，当我停止了陷入消费主义陷阱的时候，我的身体逐渐变好，我的免疫力在肉眼可见的提升，感冒也可以不药而速愈了

所以当别人再说，这个东西对男人好，那个东西对女人好，如何如何的时候，不要再轻信了，饮食有节，运动有度，过犹不及
