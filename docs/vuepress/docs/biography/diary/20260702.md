听过一句话，若以慈悲看世人，世上都是可怜人，若以因果看世人，世上没有可怜人

无神论也好 有神论也好，比较确定的事情是，
第一人类知道的有限，且受限于物理规律，物理规律就是造物者给我们的紧箍咒
第二造物者在设计我们的时候也好，或者实时监视我们也好，总是会给我们一些反馈，正向的，负向的，
有的提示温和，有的警告激烈，
有些人哪怕通过一点点温和的提示就能获取到造物者的信息，
有些人哪怕通过再激烈的警告，在人生终点也参不透造物者的意思，

还有一类是我这样的人，温和的提示没有让我警惕，直到遇到重大的挫折，才慢慢开窍开智，然后开始调整并且获得了肯定的正反馈，
然后整个人的觉知就变化了很多，
比如我之前听不见别人的声音，现在别人无心的一句话也可以是造物者借别人的口给我传递的信息，我遇到的每一件事情，
好的坏的，都是造物者给我的正向货负向的反馈，我需要调整方向以顺应大道，这可能就是老子的意思吧
老子几千年前懂得的，几千年后还有我这样的愚人，可叹
