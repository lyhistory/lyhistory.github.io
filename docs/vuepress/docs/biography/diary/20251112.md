今天一个老同学找我说要咨询点事情，她说她上小学的娃通过了某个计算机编程活动的初选，在考虑要不要花钱报名参与，因为我是计算机行业的，所以过来问我，我一时语塞，

说实话我自己搞计算机行业都是误打误撞进来的，而且我大一之前的时候根本没摸过计算机，我同学在机房打游戏，我还好奇的问一个同学，你怎么知道里面有游戏，这个d盘是啥意思。。。到大二才有了台电脑，模糊的知道了什么是计算机

我想了想，我说，首先我不了解你的娃，然后我也不敢随便替别人做决定，我也不知道现在参与是不是好的，

我同学回应说，明白，就是不干涉别人的因果，哈哈

我沉默了一会，觉着我这样说可能让同学觉着我不愿分享，然后我就想到我拿自己的娃举例，我说：
因为我的娃还小，也许等大一点我会慢慢让娃接触一些所谓编程，但是肯定是以解决问题为前提的 而且我应该不会丢给培训机构 说实话大家想法都差不多 又怕太卷又怕错过，只能根据自己的理解自己决定
比如我之前反对小孩去学轮滑，但是刚好路过小孩说想学，那就报名去学，但是比如我的小孩喜欢画画 我老婆说要报画画课，我就反对 觉着那种技术画法会抹杀小孩的创造力

同学听了表示认同，语气也轻松了起来，她也知道了我的思考过程
