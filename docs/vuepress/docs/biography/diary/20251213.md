刻板的印象只有在很强烈的刺激下才会改变，

我曾经在附近的市场买到过很老的母猪肉，所以再去买肉就非常小心的对比，比如我发现一条老街的某个年轻小伙的路边摊上卖的猪肉非常好吃，然后就一直在他这里买，直到某一天我突然意识到最近几次买的肉怎么都不好吃了，刚开始我还自我怀疑是我自己做法的问题，直到我家人也这么说，并且我家人说某大妈的猪肉很好，
但是在我印象中，某大妈的猪肉非常一般，为了验证我决定再买一次某大妈，结果确实很好，由此我得出一个结论，也许出于某些原因，那个小伙的货源变了，而且某大妈的货源也变了，然后事情就发生了转换，

所以刻板的认为我之前判断过这个小伙卖的猪肉是最好的，这个结论不可能永远成立，不要随便自我洗脑，刻板印象是很不好的习惯
