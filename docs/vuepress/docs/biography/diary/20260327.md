年轻的时候精力旺盛，没有孩子没有家庭顾虑，睡眠质量也好，人到中年，上有老下有小，精神大不如年轻时候，睡眠质量差，
因此对于房子的选择要尤其注意，下面是我总结的买房避坑指南：

1. 容易判断

周边环境：
    噪音 小区的电房设备房 | 学校广场 ，比如增城有个泉湖公园，每次去都羡慕当地村民的安置房，环境优美，直到我后来发现经常有人不分时间拿着扩音喇叭号丧式的喊歌
    污染 垃圾点 和工厂设施，最好实地走一圈，比如新塘的某个水厂，每天排放臭气熏天的污水
    辐射    周边高压线，有人说学历越低辐射越强，本人学历很高，我对这类事情没有那么确信，即使官方再辟谣，人类的认知也是有局限，比如当年欧美人把核辐射金属当做首饰佩戴
    心理不适    旁边山头很多坟 或 陵园附近

查看业主维修基金知晓物业情况

2. 有一定难度

周边如果有荒地或者破路 后续极有可能重新规划建设：高架、工厂等，只能去搜城市规划图

房子是否有贷款抵押，是否有“故事”


3. 难以判断

房子本身问题，平整 厕所防水情况 是否架空

是不是航线空域

是不是毒地块 化工厂

房子楼上楼下隔壁邻居也要关注下，昔孟母择邻处，比如我就遇到过一个痞子带着大金链，自称辽宁某地公安的东北人，一言难尽，我对东北人没意见仅对此人有意见
