今早我去父母那边，看到他们拿了一块小包装蛋糕给娃，我看了下配料表：人造奶油....

忍不住又严厉的跟我父母说这等于慢性投毒，然后我就开始想为什么新加坡的超市我印象里面没有这些东西（不代表现在，我在那边十几年没见过），
然后中国和美国的超市都大量的有这种玩意，然后我开始思考个人跟国家的利益关系，
首先声明，我爱国，我们每一代都为国家的安全付出了青春甚至是家族的生命，下面仅仅是我在和平时期的思考，跟爱国不冲突：
从博弈论的角度来说，
对国家来说，老年人是负担，年轻人是社会最需要的，所以你认为最完美的结局是怎样，我不好直接说出来，
有些东西慢性积累可能就可以达到这种效果，
新加坡则略微不同，因为太小加上生育率太低，所以他们对本国人多吃点糖多吃点盐多吃点白米饭都要加以干涉

并非谁好谁坏，无非是利益关系，

马上接近不惑年龄了，现在明白很多时候，一个东西或者一个关系都是辩证的，比如法币有时候需要持有，有时候则是被通货膨胀稀释的废纸，好的东西也有所谓垃圾时间，不能太楞
