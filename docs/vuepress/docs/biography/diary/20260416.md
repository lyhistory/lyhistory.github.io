消费主义的陷阱是一个接着一个，
我自认为已经跳脱了，结果打脸了，

比如我喜欢吃豆芽，但是外面的豆芽不知道加了什么科技，所以现在就自己发豆芽

对了关于豆子，自从我开始自发豆芽才发现有些豆子真的不能买，比如某润发的豆子直接臭掉了，只发出来1%，甚至我在增城石滩墟市买的豆子也只发了一半，一半臭了，还是买的

刚开始我买了一个自发豆芽的塑料盒，用了几次发现上面吸附了很多黄色的脏东西擦不掉，然后我就想有没有陶土的比较安全一点，

然后这几天我一直在淘宝上对比，发现有不少发豆芽的陶罐瓷罐产品，对比了几天一直没决定，今天猛然发现我不是有个煮药的陶罐吗，为什么还要买。。。

长期的消费主义让我们很难去独立思考，废物利用或者先观察手头上的工具
