病从口入

这是我最后一次清理药柜，空空的药柜里只剩下还没过期的几瓶止鼾器，看着它们我才惊觉自己被消费主义和‘现代医疗’毒害了多久，

人体是个系统，打鼾或者出现任何问题都是系统性的问题，通过所谓的止鼾器药物去阻断不仅只是治标，更恐怖的是配料表里面的化学成分在不知不觉毒害身体，产生更多的问题，

消费主义总是用兴奋的台词洗脑你，让你对它们的产品产生正向的情绪，认为选择了它们才是健康的生活健康的方式，但是它们从不主动告诉你它的产品成分，即使告知也是避重就轻，当你一次又一次的陷入到消费主义的陷阱里，
比如 青春期第一个护肤品，秋天第一杯奶茶，补肾佳品。。。。

它们是在植入你的大脑，从大脑开始控制你的身体，剥夺你本来健康的身体，当有一天你突然呕吐，急性腹泻，甚至是中风，去了医院，医生查不出问题，告诉你是小事，注意休息的时候，那是身体在警告你，你的生活方式出现了问题

现在我已经清空了家里的药柜，我的身体状态也已经达到了这几年的最好水平了，极少感冒，一两次也是两三天就自愈，大便成形，颜色正常，上下楼不再突然膝盖疼了，白天精力旺盛，晚上倒头就睡

能劝一个是一个，我爸喜欢吃零食，劝了也不听，脑梗，我的亲弟弟经常买给我爸，我就告诉我弟不要再买，我弟说他的价值观跟我不一样，觉着人活着就是怎么舒服怎么做，直到有一天他说心脏不太好，开始向我学习站桩养生，人有时候真的不碰南墙不回头，我自己不也是吗，只是有些人比较幸运早一点开悟，有些人直到生命最后一刻才后悔莫及

不要把自己变成消费主义的一个耗材，爱惜自己从入口开始
