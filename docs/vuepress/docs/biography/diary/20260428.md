这两天宝宝受凉感冒，头一晚几次因鼻塞睡着睡着做起来难受的说 我的鼻子

半夜挂了一个号

第二天想了想取消了

之前每次感冒都是第一时间跑医院，有时候是几家医院，有一次在给宝宝做小儿推拿的时候跟医师闲聊，她说儿科的主任管得很严，比如对医生开抗生素类药物设置了很高的门槛，有一次把一个医生开会当众批评了一顿，所以后面我就干脆尽量直接挂主任的号，直到有一天我看到一款儿科常用药被曝有问题而下架，但是之前主任也给我开过，刚好那段时间我开始真正的了解中医思想，我指的是以黄帝内经为主的中医思想，并且我通过自己实测，证明我通过饮食调节和作息以及一些养生技巧是完全可以摆脱药物的，我自己已经一年多没吃药了，体重自然减轻，身体自愈能力变强，然后我就开始把之前有些儿科医生跟我说的只言片语联系起来思考，比如“咳嗽本身不是问题”，“出汗就出汗，擦干就好了”，我才明白一些常见的感冒症状是身体自愈力在发挥作用的表现，身体要排毒，那么我们需要做的是帮助身体，给身体减负，而不是给身体增加负担，

一上来就吃药就是最直接的给身体增加负担的做法，不管西药中药，是药三分毒，我认可中医思想，同时知道中药博大精深，我们这些门外汉肯定不能乱用，否则适得其反，最好的做法还是这些物理减负的方式，
最起码的我们要会：看舌苔、闻口气、观察大便状态和闻味道，观察目前感冒是在风寒初期还是风寒化热，比如宝宝口臭，那么就一定要饿一顿，或者餐食减半，不要再让加重脾胃负担，身体是一个整体，免疫力需要全军作战，任何一个地方拖后腿都是不行的，白天推背，让阳气生发，白天泡脚让气血快些流动，晚上泡脚让气血回到四肢改善晚上的睡眠，晚上不要用艾草泡，就要温水即可，不要小看物理方式，比如简单的揉腹、按背，还有别人提到的滚蛋疗法，物理上在身体上的这些动作往往会带来意想不到的效果，不要轻视，我就是一开始不信走了很多弯路，不能急于求成，有时候是缓慢的，比如也许鼻塞没有立马缓解或者反复，但是随着一天天效果会越加明显。

我就是用这些办法终于实测可以让宝宝一粒药都不用吃就可以自愈

注意：我开始这样做是在娃四岁的时候，四岁之前我并没有这样试过，所以也有可能现在四岁免疫力比之前更健全更容易自愈，如果是更小的宝宝或者出现高烧一定要去医院，总之如果你想这样尝试你首先你必须有清晰的自我认知，比如你必须是做事严谨的父母，否则还是交给专业的医生。
