我自己的原生家庭，父母经常吵架打架并且掀桌子甚至当着面说要动刀子，
从小活在恐怖的氛围中，我暗自下决心要给自己的小孩足够安全感，足够的克制自己的情绪，
因为这种家庭环境长大的我时不时有内心的愤怒，尤其是遇到事情的时候很容易大动肝火，
有时候宝宝无理取闹，我也忍耐性子不会动怒，并且我自我感觉良好，觉着自己做的不错，

说了句难听的话，头扭到一边，不理宝宝，宝宝虽然还小，但是好像意识到了，拉扯了我几下就乖乖去睡了，
嘴里还嘟嘟囔囔的小声说着啥，
我又在想我下次不能这样，可以扭头到一边，千万不能再说那种难听的话了，否则跟我父母有啥区别，
又有一次给宝宝报的网课，第一次上网课的过程中小动作不断，根本不听老师讲，即使老师放的吸引人的动画短片，
