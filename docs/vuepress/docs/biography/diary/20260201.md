真遗传和假遗传

二十多岁的时候我曾经觉着我的学历比我老婆高，工资也比她多，所以她说的话我从来听不进去，
三十多岁我才发现我老婆曾经的预判一一应验，她对人的判断，事情的判断可以说准确无误，
然后我就开始想为什么会这样，一直想不通，
然后我只是按照她说的一些方式去观察别人，慢慢的我发现我也学会了判断，
曾经我老婆说这些都是遗传的，
但是我自己的改变直接否定了这个说法，我认为是假遗传，
我是小镇做题家，全国贫困县，县里只有面粉厂和火葬厂，打工都没地方，所有的邻居老师都是告诉我，高考是唯一出路，所以我不断的做题，我的大脑突触神经链路经过训练以及形成了固化的电路板，适配考试，
然后大学到工作都是从事IT技术，大脑依然是沿着做题的路径强化，
直到我这几年经历过了一些挫折，对健康、家庭、社会关系有了不同的理解，才开始学着按照我老婆的方式思考，
然后我的大脑就慢慢的形成了新的链接，所以我突然发现大脑会自动帮我分析，很多时候都是事后脑子里放电影出现的结论和发现，这在以前是不可能的，因为以前我并不会留意除了做题之外的人性分析，

所以我告诉我老婆，这次你错了，这种看人的本领并不是遗传，而是因为我的父母通过他们的言行把他们的那套思考方式刻录到了我的大脑，然后我有样学样的，看起来像遗传，其实可以改变，但是不容易，需要外界刺激或者自我的觉醒才能改变

“某一天，我发现了一个惊天的秘密，就是那些伟大的人物之所以那么轻松地就成就了他们的伟大，不只是因为聪明，最重要的是人家很早的时候就念过正经的、有学问的书。他们生来注定要伟大，他们所受的教育指向伟大，在成长的岁月里他们一直有伟大的自觉。” -- 《云端脚下 - 从一元二次方程到规范场论》
