这个年代要活得好就必须学会判断，但是会判断真的很难，

比如很多人都说牛奶有营养，但是喝牛奶真的好吗？我们先说两个故事：

故事1： 培根和鸡蛋成为早餐标配，很大程度上是一场经典的公关营销案例。

背景：20世纪20年代，美国人的早餐习惯趋于清淡，培根生产商比纳公司（Beech-Nut）面临产品滞销的压力。

策略：公关专家爱德华·伯内斯（Edward Bernays）受雇解决这一问题。他巧妙地利用其舅舅西格蒙德·弗洛伊德的理论，操纵公众心理。

执行：伯内斯策划了一项活动，他声称征求了专业医生的意见，认为丰盛的早餐（特指包含培根和鸡蛋）比清淡的早餐更有益健康。他成功动员了数千名医生签署支持这一观点的“请愿书”，并将其作为“科学研究”在报纸上发表。

结果：这场营销活动极大地改变了公众的观念，培根和鸡蛋的组合迅速被接受为理想早餐，培根的销量也随之大增。

故事2： 即食麦片的发明和推广则与一场带有清教色彩的健康运动有关。

起源：19世纪中后期，基督复临安息日会的成员在疗养院中推广健康饮食。詹姆斯·杰克逊发明了最早的谷物早餐“Granula”，约翰·哈维·凯洛格博士则发明了玉米片。

动机：凯洛格等人推广清淡的谷物早餐，部分原因是他们认为这有助于抑制性欲，符合其宗教道德信条。

商业化：真正让麦片广泛流行起来的，是其无与伦比的便利性，非常适合工业时代快节奏的生活。后来，商家为了销售，开始在广播中大肆宣传“早餐是一天中最重要的一餐”的观念，并将他们的麦片产品包装成最健康的选择。战后女性就业率增加，需要快速准备早餐，也进一步促进了即食麦片的消费。
