我对面邻居家有几个孩子，大女儿听说也不工作，天天在家玩，二女儿上中学，小儿子上小学，
我跟他们几个都打过招呼，大女儿基本不理人，二女儿最有礼貌，有些怯懦，还算正常，小儿子感觉就是很丧，
每次都是有气无力无精打采，带着跟这个年龄不匹配的眼镜，完全没有少年的活力

我想起了自己的少年，那时候虽然学业很重，但是充满精气神

现在的小学生要上晚自习到很晚，虽然我看他周末父母会陪着打打羽毛球骑自行车，但是感觉灭掉的火苗很难再燃

这样的精气神去学习有什么意义吗，不要说有原始的好奇心和冲动了，如果没有自己的想法，最终在后面漫长的学习道路上走歪路是很自然的事情，
最好的结局就是成为一个肥胖的中年男人，只能通过抽烟喝酒排解烦恼吧
