虽然我在广州郊区生活，但是所在地却是发展快速，全国排名靠前的镇子，交通比较便利，最近小区的租户逐渐增多，
并且由于是郊区，租户素质堪忧，索性我决定把房子出租，然后搬到附近一个村子居住，村子租金很低，这样每个月还有放租的盈利到手，
这个村子坐落于一个水库旁，环境优美，村民和善，等我住进来我也有了一些有意思的发现和事情发生

 本来以为这个村子跟我北方农村一样安静，我白天来的时候确实如此，不过晚上睡觉第一天就听到了嗡嗡声，刚开始以为是谁家的空调外机，
后来才发现是远处的高速路，打开地图才发现这个村子四面都被高速路包围，不过还算空气比较好，就当白噪音，住了几天也习惯了

然后就是今天早上起来遇到件奇葩的事情，早起去吃早餐，楼下经过一个老伯，他喊住我又跟我寒暄，听我说去吃早餐，拉着我去旁边的早餐店请我吃肠粉，
我以为他是本地人，结果他是另一个村的，我满心疑惑他这么热情，慢慢才知道原因，他说觉着我五官很上镜，他每天开直播，
今天碰到我就想开直播跟我聊天直播，说着就举起手机，我赶忙阻拦说不方便，然后他只好作罢，他有给我看了他的直播：
一张图片写着什么AI颠覆人类文明之类的网文，背景乐是笛子还是什么玩意的合奏，背景乐跟说话声混杂一起感觉像在听传销。。。

然后他开始跟我聊一些完全没有营养的情绪化的观点，什么现在教育不行，IT行业不行，马化腾如何如何亏钱，
我听了两句就知道此地不宜久留，抓紧吃完了肠粉，托词离开

能量不符，完全不是一个世界的人，思考的深度鸡同鸭讲，不知道为啥现在很多他这样的人就喜欢各种新词，拾人牙慧，言之无物，说着不知所云的话，甚至不知道自己在说啥，

我突然想起来我的一个厦大的学长，他十几岁研究密码学，钻研别人看不上的底层基础知识，非常简单的一句话都够我思考一周

想到这里，下次再遇到此类人要尽快脱身，否则每一分钟都是对自己能量的浪费
