### 泡脚
儿童足浴的通用推荐就是睡前：多家儿童医院（济南儿童医院、泰安中医医院）都明确说儿童泡脚最好晚上8点左右、睡前1-2小时（19-21点），这时候肾经气血旺，药效吸收好，还能安神助睡。
不要早上泡：苗苗刚起床时咽喉最干最敏感，早上泡完去幼儿园/出门容易受风，反而刺激咳嗽。
不要中午泡：饭后马上泡抢血供影响消化，苗苗舌苔刚薄下来，脾胃还在恢复，别折腾。
饭后一小时再泡
泡后立刻保暖
单次时长：5 - 10分钟。
水温控制：38℃ - 40℃（温热即可）。
水位高度：没过脚踝（三阴交穴）即可。

鼻窒方：治病通关包（夏枯草、白芷、路路通、浙贝母、皂角刺、连翘、辛夷、炒苍耳子）
核心功效：清热化痰、消肿通窍。

适用时机：“有症状”时用（主攻期）。

什么时候泡：苗苗今早喉咙疼、咳嗽有痰、感觉呼吸道还有“垃圾”没排干净的时候。

为什么要泡：方子里的浙贝母能帮她化掉最后那点顽固的痰，夏枯草能清热，白芷和苍耳子能通鼻窍。这包药是用来帮她把身体里的“火”和“痰”彻底清出去的。

特禀方：“体质防护盾”（黄芪、白术、桂枝、防风、醋五味子、乌梅、银柴胡。）
核心功效：益气固表、调和营卫、抗过敏。

适用时机：“没症状”时用（防守期）。

什么时候泡：等苗苗完全不咳、不疼、不热了，只剩下“容易出汗、体质偏弱、怕风怕冷”这些底子问题的时候。

为什么要泡：黄芪、白术、防风这三味药是著名的“玉屏风散”底子，能帮孩子把体表的“围墙”修牢固，以后就不容易受风寒侵袭，也不容易反复呼吸道感染了。乌梅和银柴胡还能起到抗过敏、收敛的作用。

### 滚蛋
滚蛋疗法在儿科应用广泛，尤其适合以下常见病症：


1. 感冒咳嗽：一名4岁男童，反复咳嗽3个月，经西医诊断为慢性咳嗽。在常规西药治疗基础上，中医科医生在患儿头部风池、印堂，背部膀胱经，胸部云门、中府、膻中、天突等穴位进行滚蛋治疗。经过3次治疗，患儿咳嗽症状大大缓解，睡眠状况也明显好转。


2. 腹痛腹泻：临床研究表明，中药滚蛋疗法联合西药治疗小儿腹泻，总有效率可达90.54%，能显著缩短止泻、退热及大便性状恢复正常的时间。


3. 小儿夜啼惊风：孩子晚上哭闹不停，可用温热鸡蛋先从头面部开始，接着正面从上身滚至下身，然后是背部从上身滚至下身，最后从脚心"收"出去。此法有安神定惊的作用。


4. 眼部浮肿：用温热鸡蛋（约40℃）去壳后，用蛋白部分轻滚眼周，从内眼角向太阳穴方向滚动，避开眼球，每次5分钟，可促进眼周血液循环。


具体操作方法


材料准备：


1. 新鲜鸡蛋1-2个（不可用变质蛋）

2. 根据需要准备中药材：风寒证可用桂枝、生姜、艾叶等发散风寒的药材；脾虚腹泻可用党参、白术、茯苓等健脾药材。

3. 纱布、纯银片（可选，银器传统认为有吸附驱邪作用）


操作步骤：


1. 将鸡蛋和药材一起煮15-30分钟，至鸡蛋完全煮熟。

2. 捞出鸡蛋，剥去外壳，去掉蛋黄（可选），加入纯银片（可选），用纱布包裹。

3. 待温度适宜（儿童以60-70℃为宜，操作前务必以手背试温，避免烫伤），在相应穴位或部位滚动。

4. 滚动时手法要轻柔，像揉面团一样轻滚，不要用力按压。

5. 蛋凉后可再放入药液中加热，继续滚动，反复多次，直至皮肤微微出汗或局部轻微红晕为止。


穴位选择：


· 感冒：大椎穴、印堂穴、风池穴和风府穴

· 咳嗽：肺俞穴、云门穴、中府穴、天突穴

· 腹痛腹泻：以肚脐为中心的腹部

· 鼻塞鼻炎：前额、印堂、迎香穴、膀胱经

重要注意事项

1. 温度控制：儿童皮肤娇嫩，温度控制在60-70℃为宜，操作前务必用手背试温。

2. 使用后的鸡蛋：切勿食用！滚过的鸡蛋不仅吸收了病邪之气，还沾染了皮肤上的汗液、皮脂等，已经受到污染，应立即丢弃。

3. 禁忌人群：

   · 对中药或鸡蛋过敏者

   · 皮肤有破损、湿疹、疖肿者

   · 伴有心脏病或出血倾向者慎用

   · 各种传染病、结核性、感染性疾病或恶性肿瘤、骨折、脱位等患者

4. 治疗后护理：治疗后4小时内避免吹风，不能洗冷水澡，注意保暖。

### 食疗

鼻塞：吃柠檬 如果孩子症状不明显，只是鼻子堵，不通气，家长可以用新鲜柠檬泡温水给孩子喝。柠檬味酸，具有收敛的作用，能够减轻鼻黏膜的充血和肿胀情况，使鼻腔通道变宽，让鼻腔感觉更加通畅

流鼻涕：吃小米 孩子遇风寒邪气容易鼻涕不断，是脾胃虚弱，抵抗力差，可以喝小米粥养脾胃，煮粥时可加入山药、黄芪一起煮。小米健脾和胃，山药健脾益肺，黄芪益气固表，煮粥喝不仅能把孩子脾胃慢慢养好，还能增强抵抗力，让孩子有能力抵御外邪入侵。

咳嗽：吃杏仁 如果孩子鼻炎，伴有咽干咳嗽的情况，可以吃杏仁缓解咳嗽。杏仁能止咳平喘、润肺清火，家长可以在煮粥时加入杏仁同煮食用，也可以和其他食材一起打成杏仁糊食用。

怕冷：吃核桃 孩子畏寒怕冷、阳气不足，可以吃核桃，生吃或做养生餐都可以。核桃能补“命门之火”，把阳气补到根上，引气归元，不仅能让孩子全身温暖起来，还能增强抵抗力，不再惧怕严寒。

### 护眼
中医认为，眼与全身脏腑和经络的联系密切，古代医学家根据临床实践，总结了许多简便而有效的养睛明目的方法，现介绍几种眼保健法如下：

按摩护眼方法：

熨目法：黎明起床，先将双手互相摩擦，待手搓热后一手掌熨贴双眼，反复三次以后，再以食、中指轻轻按压眼球，或按压眼球四周。

运目法：两脚分开与肩宽，挺胸站立，头梢仰。瞪大双眼，尽量使眼球不停转动（头不动），先从右向左转10次，从左向右转10次。然后停，放松肌肉，再重复上述运动，如此3遍。此法于早晨在花园内进行最好，能起到醒脑明目之功效。

低头法：身体取下蹲姿式，用双手分别攀住两脚五趾，并稍微用力地往上扳，用力时尽量朝下低头，这样便有助于使五脏六腑的精气精气上升至头部，从而起到营养耳目之作用。

吐气法：腰背挺直坐位，以鼻子徐涂吸气，侍气吸到最大限度时，用右手捏住鼻孔，紧闭双眼，再用口慢慢地吐气。

折指法：每天坚持早晚各做1遍小指向内折弯，再向后搬的屈伸运动。每遍进行30～50次并在小指外侧的基部用拇指和食指揉捏50-100次。此法坐、立、卧皆可做，坚持经常做，不仅能养脑明目，对有白内障和其它眼病者也有一定疗效。

以上诸法可以单独做，也可任选1、2种合做，贵在持之有恒，日久定见成效。

护眼茶饮小品：

1、芝麻枸杞茶：

材料：沙苑子 10克、菟丝子10克、黑芝麻12克、枸杞子20克、首乌15克、泽兰10克、食盐10克。

作法：食盐炒热，加入枸杞续炒至发胀，即筛去食盐取枸杞与余药煎煮约 12分钟，再浸泡10分钟，滤去渣，代茶饮用。

功效：明目。主治视力减退。

2、五味蜜茶：

材料：北五味 4克、蜜蒙花6克、绿茶粉1克、蜂蜜10克。

作法：北五味子入锅略炒稍焦，再加入蜜蒙花与水约 450C.C.，煮沸3分钟，过滤加蜂蜜拌匀，待稍冷适量时再入绿茶粉搅和均匀即可饮用。

功效：主治视力功能减退。

3、决明子茶：

材料：决明子 10克、菊花5克、山楂15克。

作法：决明子略捣碎后，加入菊花、山楂，以沸水冲洗，加盖焖约 30分钟，即可饮用。

功效：主治头部晕眩，目昏干涩，视力减退。

家庭护眼药膳：

对于眼睛的保健，除了多接近大自然，多看远处及绿色植物，日常生活中也要让眼睛有足够的休息时间。此外，饮食中应适当补充富含维生素 A的食物，如红萝卜、海藻、绿色蔬菜、鱼肝油、动物肝脏等，因维生素A缺乏时易引起视觉障碍、眼睛疲劳、眼屎多、角膜红肿等症。中药方面，可服用六味地黄丸加枸杞、菊花，或柴胡加龙骨牡蛎汤，均可有效改善视力衰退、眼睛疲劳等症。提供三款药膳：

一、养血护眼汤：

材料：鸭肝 150克、芹菜50克、木耳（水发）20克、鲜蘑菇50克、蒜头、葱姜、麻油、盐、米酒各适量。

制作方法：

1.芹菜切小段，磨菇、木耳切丁，肝切成泥状拌入米酒、盐及太白粉。

2.油烧至5分热，爆香姜、蒜，加入蘑茹、木耳，加适量水，煮沸，再加入肝泥并搅和，调味，投入芹菜煮沸，起锅滴入少许麻油。

功效：补益肝肾、养血明目、增强免疫力。适合肝虚目暗、视物模糊、夜盲贫血症，是小儿护眼的最佳食疗方。

二、补肝护眼汤：

材料： 1.猪肝60克或土鸡肝2付、菠菜130克、食盐、香油各少许，清高汤1公升。

2.故纸、谷精、甘杞、川芎各15克。

作法： 1.将四味中药材洗净加水1000C.C.，煎煮约20分钟，滤渣留汤备用。

2.猪肝去筋膜洗净后切薄片，菠菜洗净后切成小段备用。先用少量油爆香葱花，加入中药汁、猪肝、菠菜，煮开后放入适量食盐，搅均后起锅加入少许香油即可食用。

功效：补肝养血、明目润燥。常食可改善视力，并可治疗小儿夜盲症、贫血症，均有良好的补益作用

4. 咳嗽
    陈兆鸿 主任医师
    流鼻涕咳嗽是受凉引起呼吸道感染，要加厚衣服保暖才能好转。
    急性上呼吸道感染通常是由于受凉后呼吸道的免疫力降低而发病，受凉后呼吸道分泌物（鼻涕和痰）就会增多，这些分泌物刺激呼吸道粘膜会引起咳嗽，只有不再让小孩受凉才会好转。
    反复受凉就很难好转。要穿厚衣服保暖，感觉背后皮肤有微汗才算够暖，放汗巾在背后吸汗，这样就可以避免受凉了。
    小孩晚上睡觉时容易踢被子而受凉，要穿厚的马甲睡觉，保持背后皮肤有汗，放汗巾在背后吸汗，这样就可以避免受凉了。

夜间人体迷走神经张力升高，会导致 ​气道平滑肌收缩、黏膜分泌增多，本身就容易引发咳嗽。若咽喉/气道黏膜对过敏原敏感（即“喉咙过敏”），这种“收缩+分泌”的刺激会被放大，导致夜间咳嗽更显著。

​冷空气​：夜间室温下降，空气刺激气道黏膜，诱发过敏样收缩。

​其他潜在过敏原​：如宠物皮屑、霉菌（若环境中有此类暴露，夜间更易接触）。

3. 支气管炎本身的“炎症叠加”
急性支气管炎的炎症会让气道黏膜处于“高敏感状态”，夜间迷走神经兴奋+过敏原刺激，会进一步放大咳嗽反射，导致“白天偶尔咳、夜间频繁咳”。

一、病情发展过程梳理（中医视角）

患儿初始表现为打喷嚏、鼻塞、流清涕、偶有咳嗽，结合体格检查“咽微红、舌淡红苔薄白、脉弦细”，中医辨证为风寒袭肺证（初期外邪犯表，肺卫失和）。

第一阶段（初期）：外邪（风寒）侵袭皮毛，肺开窍于鼻，肺气失宣，故打喷嚏、鼻塞、流清涕；肺失肃降，气逆于上，故偶有咳嗽。此时病情较轻，属“表证”。

第二阶段（用药后变化）：服用第一剂中药（含较多辛温解表药如白芷、苍耳子，及宣肺药如苦杏仁、桔梗）后，出现夜间亢奋、鼻血、咳嗽加重。结合后续检查“咽稍充血、扁桃体Ⅰ度肿大、双肺呼吸音粗”，提示病情从“风寒袭肺”向“外寒里热”转化（即风寒之邪未散，入里化热，热邪熏蒸咽喉、气道，导致咳嗽加重、咽充血；热迫血妄行，故流鼻血；辛温药可能加重热象，导致夜间亢奋）。

第三阶段（调整后）：停泡脚（避免进一步温热助邪）、调整药量（减少辛温药用量）后，咳嗽仍未缓解，但次日检查发现扁桃体肿大、呼吸音粗，说明里热加重。此时换用小儿荆杏止咳颗粒（疏风散寒、宣肺清热），一天后咳嗽大大减轻，提示里热得清，肺复宣降。

二、第一剂中药无效的原因分析

第一剂中药无效并非“药不对症”，而是病情进展与药物适配性变化所致，具体原因如下：

药物组成与病情进展不匹配：

第一剂中药以辛温解表、宣肺通窍为主（如白芷、苍耳子、辛夷、紫苏叶），适用于初期“风寒袭肺”证。但随着病情发展，患儿已从“表寒”转为“外寒里热”（咽充血、扁桃体肿大），此时需兼顾散寒与清热，而第一剂中药中清热之力不足（仅浙贝母、木蝴蝶略有清热），无法应对里热加重的情况，故咳嗽加重。

辛温药用量偏大，加重热象：

第一剂中药中白芷（10g）、苍耳子（5g）、紫苏叶（5g）均为辛温之品，虽能解表，但过量易助热化火。患儿服用后，辛温药可能加重里热，导致热邪熏蒸咽喉（咽充血）、气道（咳嗽加重）、迫血妄行（鼻血），反而使病情恶化。

泡脚的温热效应叠加：

患儿同时用“鼻鼽包”泡脚（成分含艾叶、紫苏叶等辛温药），温热刺激可能加重体内热象，与中药的辛温作用叠加，进一步加剧“里热”，导致咳嗽加重、鼻血。

三、第二剂药（小儿荆杏止咳颗粒）有效的原因

小儿荆杏止咳颗粒的成分（荆芥、矮地茶、蜜麻黄、苦杏仁、黄芩、前胡等）正好匹配患儿“外寒里热”的病机，故能快速起效，具体机制如下：

疏风散寒，针对“外寒”：

成分中的荆芥、蜜麻黄能发散风寒，解除表邪（如鼻塞、流清涕），符合“外寒”的病机。

宣肺清热，针对“里热”：

成分中的黄芩、前胡、紫菀能清泻里热（如咽充血、扁桃体肿大），同时宣肺止咳（如咳嗽加重）。其中，黄芩是清热的关键药物，能有效清除肺热，缓解热邪对气道的熏蒸；前胡、紫菀则能宣通肺气、化痰止咳，改善咳嗽症状。

兼顾肺气宣降，恢复肺功能：

成分中的苦杏仁、桔梗能降气止咳（苦杏仁）、宣肺利咽（桔梗），两者配合，恢复肺的“宣发肃降”功能，从根本上解决咳嗽问题。

四、总结：病情发展与治疗的中医逻辑

患儿的病情发展符合“外感病从表入里、从寒化热”的规律（初期风寒袭表→中期外寒里热→后期里热渐清）。第一剂中药无效是因为未能及时应对“化热”的病情进展，而第二剂药（小儿荆杏止咳颗粒）有效是因为精准匹配了“外寒里热”的病机，兼顾了散寒与清热、宣肺与肃降。

关键启示：中医治疗感冒需“随证变法”（根据病情进展调整药物），而非一成不变。初期风寒证用辛温解表，中期化热则需“散寒清热并用”，后期则需“清余热、补肺气”。同时，儿童用药需注意“中病即止”（避免过量使用辛温药），并密切观察病情变化，及时调整治疗方案。

儿童感冒的常见发展规律是：初期为“纯风寒”（或“风热”）→ 若未及时干预，寒邪（或热邪）入里化热→ 发展为“外寒里热”或“纯热”证。
