适合：4岁

⑤：Bee-Bot
适合：6岁 
特色：适合有编程基础的，思考方式更加丰富，支持断点续玩，更接近真实编程过程。

适合：6岁

适合：6岁 
特色：苹果官方开发，使用流畅，教学指导更加精致，结合图形与编码，难度较高但更贴近真实编程。

适合：7岁 
特色：三维场景，重在循环指令，支持调试，适合进阶学习阶段。
