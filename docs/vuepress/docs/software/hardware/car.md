## ECU - Engine control unit 发动机控制器
是一种控制内燃机各个部分运作的电子装置。 最简单的ECU只控制每个引擎周期的注油量。 在现代汽车上配备的更高级的发动机控制器还控制点火时间、可变阀门时间（VVT）、涡轮增压器维持的推进级别（配备涡轮增压的汽车）和其他外围设备。
